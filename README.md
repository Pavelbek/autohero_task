# Test task

## Technologies

* Programming language: Java 1.8
* Build tool: Gradle 5.0
* Automation framework: Selenide
* Reporting tool: Allure 2

## How to:

1. Go to test-task folder
1. Run: ./gradlew clean
1. Run: ./gradlew ui-tests:test
1. Run: ./gradlew ui-tests:allureReport

**Report is located in: ui-tests/build/reports/allure-report. Open index.html in browser to see the report.**
